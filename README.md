# Introdução

Projeto feito para organização de leitura pessoal.

Nesse projeto você pode cadastrar seus livros.

O cadastro terá as seguintes informações: Título do livro, nome do autor, gênero do livro e o status que a leitura se encontra.

Você também pode ver todos os livros cadastrados.

Também pode atualizar alguma informação que esteja errada ou mudar os status que a leitura se encontra.

E claro, também existe a opção de excluir o livro.
